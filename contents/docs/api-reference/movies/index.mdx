## Available methods

- [Details](/docs/api-reference/movies/details): Get detailed metadata about a movie.
- [Credits](/docs/api-reference/movies/credits): Fetch cast and crew information for a movie.
- [Alternative Titles](/docs/api-reference/movies/alternative_titles): Get localized and alternative titles for a movie.
