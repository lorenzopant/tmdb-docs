---
title: Translations
description: Get the translations for a movie.
---

Take a read through the [TMDB language documentation](https://developer.themoviedb.org/docs/languages) for more information about languages on TMDB.

```typescript
tmdb.movies.translations(params: MovieTranslationsParams): Promise<MovieTranslations>; 
```

## Parameters

| Name      | Type   | Required | Description                                      |
|-----------|--------|----------|--------------------------------------------------|
| movie_id  | number         | ✅      | The ID of the movie.                             |

<Note title="Reference">
    <p>[https://developer.themoviedb.org/reference/movie-translations](https://developer.themoviedb.org/reference/movie-translations)</p>
</Note>


## Example

```typescript
const result = await tmdb.movie.translations({ movie_id: 550 });
```

## Types

### Response Type

```typescript
export type MovieTranslations = {
	id: number;
	translations: MovieTranslationResults[];
};
```

### Related Types

```typescript
export type MovieTranslationResults = {
	iso_3166_1: string;
	iso_639_1: string;
	name: string;
	english_name: string;
	data: {
		homepage: string;
		overview: string;
		runtime: number;
		tagline: string;
		title: string;
	};
};

export type MovieTranslationsParams = {
	movie_id: number;
};
```
