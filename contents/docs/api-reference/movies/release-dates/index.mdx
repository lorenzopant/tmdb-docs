---
title: Release Dates
description: Get the release dates and certifications for a movie. For different countries and release types.
---
 
The release types and statuses used on TMDB are the following:

| Type | Description           |
|------|-----------------------|
| 1    | Premiere              |
| 2    | Theatrical (Limited)  |
| 3    | Theatrical            |
| 4    | Digital               |
| 5    | Physical              |
| 6    | TV                    |

```typescript
tmdb.movies.release_dates(params: MovieReleaseDatesParams): Promise<MovieReleaseDates>; 
```

## Parameters

| Name      | Type   | Required | Description                                      |
|-----------|--------|----------|--------------------------------------------------|

## Example

```typescript
const result = await tmdb.movie.release_dates({ movie_id: 550 });
```

## Types
