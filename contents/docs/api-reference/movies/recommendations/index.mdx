---
title: Recommendations
description: Get a list of recommended movies for a specific movie.
---
 
This is based on the movie's popularity and user ratings.

```typescript
tmdb.movies.recommendations(params: MovieRecommendationsParams): Promise<MovieRecommendations>; 
```

## Parameters

| Name      | Type   | Required | Description                                      |
|-----------|--------|----------|--------------------------------------------------|
| movie_id  | number         | ✅      | The ID of the movie.                             |
| page   | number | ❌       | Page number of the results. Defaults to 1. |
| language   | LanguageISO6391 | ❌       | Language for the response |

<Note title="Reference">
    <p>[https://developer.themoviedb.org/reference/movie-recommendations](https://developer.themoviedb.org/reference/movie-recommendations)</p>
</Note>


## Example

```typescript
const result = await tmdb.movie.recommendations({ movie_id: 550 });
```

## Types

### Response Type

```typescript
export type MovieRecommendations = PaginatedResponse<MovieResultItem>;
```

