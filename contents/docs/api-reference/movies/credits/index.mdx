---
title: Credits
description: Get the credits for a movie (e.g., crew, director, actors etc.).
---
 
```typescript
tmdb.movies.credits(params: MovieCreditsParams): Promise<MovieCredits>; 
```

## Parameters

| Name      | Type   | Required | Description                                      |
|-----------|--------|----------|--------------------------------------------------|
| movie_id  | number         | ✅      | The ID of the movie.                             |
| language   | LanguageISO6391 | ❌       | Language for the response |

<Note title="Reference">
    <p>[https://developer.themoviedb.org/reference/movie-credits](https://developer.themoviedb.org/reference/movie-credits)</p>
</Note>


## Example

```typescript
const result = await tmdb.movie.credits({ movie_id: 550 });
```

## Types

### Response Type

```typescript
export type MovieCredits = {
	id: number;
	cast: Cast[];
	crew: Crew[];
};
```
