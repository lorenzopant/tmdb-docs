---
title: Watch Providers
description: Get the list of streaming providers we have for a movie.
---

Powered by our partnership with JustWatch, you can query this method to get a list of the streaming/rental/purchase availabilities per country by provider.
This is not going to return full deep links, but rather, it's just enough information to display what's available where.

<Note title="JustWatch Attribution Required" type="warning">
Following [TMDB Guidelines](https://developer.themoviedb.org/reference/movie-watch-providers): In order to use this data you must attribute the source of the data as JustWatch. If we find any usage not complying with these terms we will revoke access to the API.
</Note>

```typescript
tmdb.movies.watch_providers(params: MovieWathProvidersParams): Promise<MovieWatchProvider>; 
```

## Example

```typescript
const result = await tmdb.movie.watch_providers({ movie_id: 550 });
```

## Types

### Response Type

```typescript
export type MovieWatchProvider = {
	id: number;
	results: Record<CountryISO3166_1, WatchProvider[]>;
};
```
