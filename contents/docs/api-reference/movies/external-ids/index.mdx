---
title: External IDs
description: Get the external IDs for a movie, supported external IDs are IMDB, Facebook, Instagram, Twitter, and Wikidata.
---
 
```typescript
tmdb.movies.external_ids(params: MovieExternalIDsParams): Promise<MovieExternalIDs>; 
```

## Parameters

| Name      | Type   | Required | Description                                      |
|-----------|--------|----------|--------------------------------------------------|
| movie_id  | number         | âœ…      | The ID of the movie.                             |

<Note title="Reference">
    <p>[https://developer.themoviedb.org/reference/movie-external-ids](https://developer.themoviedb.org/reference/movie-external-ids)</p>
</Note>


## Example

```typescript
const result = await tmdb.movie.external_ids({ movie_id: 550 });
```

## Types

### Response Type

```typescript
export type MovieExternalIDs = {
	id: number;
	imdb_id: string | null;
	facebook_id: string | null;
	twitter_id: string | null;
	instagram_id: string | null;
};
```

### Related Types

```typescript
export type MovieExternalIDsParams = {
	movie_id: number;
};
```
