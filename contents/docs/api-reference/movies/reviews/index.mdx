---
title: Reviews
description: Get the users reviews for a movie.
---
 
```typescript
tmdb.movies.reviews(params: MovieReviewsParams): Promise<MovieReviews>; 
```

## Parameters

| Name      | Type   | Required | Description                                      |
|-----------|--------|----------|--------------------------------------------------|
| movie_id  | number         | ✅      | The ID of the movie.                             |
| page   | number | ❌       | Page number of the results to return. Defaults to 1. |
| language   | LanguageISO6391 | ❌       | Language for the response |

<Note title="Reference">
    <p>[https://developer.themoviedb.org/reference/movie-reviews](https://developer.themoviedb.org/reference/movie-reviews)</p>
</Note>


## Example

```typescript
const result = await tmdb.movie.reviews({ movie_id: 550 });
```

## Types

### Response Type

```typescript
export type MovieReviews = PaginatedResponse<MovieReview>;
```
