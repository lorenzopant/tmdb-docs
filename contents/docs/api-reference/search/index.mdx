## Search Movies 

`tmdb.search.movies`
Search for movies by title, keyword, or phrase.

```typescript
tmdb.search.movies(options: {
  query: string;
  page?: number;
  language?: string;
  include_adult?: boolean;
  region?: string;
  primary_release_year?: string;
  year?: string
}): Promise<PaginatedResponse<MovieResultItem>>
```

### Parameters

| Name                       | Type      | Required | Description                                           |
|----------------------------|-----------|----------|-------------------------------------------------------|
| `query`                    | `string`  | ‚úÖ       | Search query text                                     |
| `include_adult`            | `boolean` | ‚ùå       | Whether to include adult content (default: false)     |
| `language`                 | `string`  | ‚ùå       | Language code (default: `en-US`)                      |
| `primary_release_year`     | `string`  | ‚ùå       | Primary release year of the movie                     |
| `page`                     | `number`  | ‚ùå       | Page number (default: 1)                              |
| `region`                   | `string`  | ‚ùå       | Region code (ISO 3166-1)                              |
| `year`                     | `string`  | ‚ùå       | Release year of the movie                             |


<Note title="TMDB Reference">
[https://developer.themoviedb.org/reference/search-movie](https://developer.themoviedb.org/reference/search-movie)

### Example

```ts
const results = await tmdb.search.movies({ query: "Inception" });
```

### Types
```typescript
type MovieResultItem = {
	backdrop_path: string;
	id: number;
	title: string;
	original_title: string;
	overview: string;
	poster_path: string;
	media_type: string;
	adult: boolean;
	original_language: string;
	genre_ids: number[];
	popularity: number;
	release_date: string;
	video: boolean;
	vote_average: number;
	vote_count: number;
};

type PaginatedResponse<T> = {
	page: number;
	total_pages: number;
	total_results: number;
	results: T[];
};
```
