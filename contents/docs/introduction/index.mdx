Welcome to **@lorenzopant/tmdb Docs**! This site documents a fully type-safe, modern TypeScript SDK for [The Movie Database (TMDB)](https://developer.themoviedb.org/docs). Whether you're building a movie app, integrating theater listings, or creating a personal watchlist tracker, this SDK offers a clean and intuitive API for working with TMDB data.
